Carousel
========

openframeworks app to control a KodakCarousel using ofxDfRobotRelayShield for Arduino

Carousel connection(check the photo from /schematics):
- pin 1 forward
- pin 2 backward
- pin 3 common

Dependencies:
- ofxXmlSettings
- ofxDfRobotRelayShield
- ofxMSATimer
- ofxRange
- ofxTextInputField
- ofxTimecode
- ofxTimeline
- ofxTimer
- ofxTween
- ofxUI
